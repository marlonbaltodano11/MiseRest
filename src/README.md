# Source Code (`src/`)

All application source code lives here. This folder separates source code from configuration and project root files.

## Main folders:

- **app/**: App Router pages and API routes.
- **components/**: Reusable UI components.
- **styles/**: CSS/SCSS files.
- **lib/**: Helpers, utilities, and API clients.
- **hooks/**: Custom React hooks.
- **context/**: React context providers.
- **types/**: TypeScript global types.
